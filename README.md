## 🚀 Getting Started

### 1. Clone the repository
```bash
git clone https://github.com/your-username/matplotlib-visualization.git
cd matplotlib-visualization
```

### 2. Install dependencies
```bash
pip install matplotlib numpy pandas jupyter
```

### 3. Run the notebook — this generates all chart images into `assets/`
```bash
jupyter notebook matplotlib_portfolio.ipynb
```

> All plots are saved to the `assets/` folder automatically when you run the notebook. GitHub will then display them here in the README.

## ✨ Design Choices

All charts share a unified visual theme:
- White background with subtle `#F9F9F9` axes panel
- Dashed light-grey grid lines — present but unobtrusive
- Top & right spines removed for a cleaner look
- Consistent muted colour palette across all charts
- Value labels on bar charts for immediate readability
